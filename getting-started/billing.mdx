BrewBot offers both free and premium tiers.

## Plans

| Plan | Price | Details |
|------|-------|---------|
| Free | $0 | Reaction roles, welcome messages, mod logs, mod agreements |
| Monthly | $25/month | All features |
| Yearly | $249/year | All features, save ~17% |
| Setup fee | $149 one-time | Required for new premium subscriptions |

## Free features

The following features are available on the free plan:

- Reaction roles
- Welcome messages
- Mod logs
- Mod agreements

## Premium features

Premium unlocks the full feature set:

- Bot personalizer
- Custom commands
- Social alerts
- Leveling
- Live monitor
- Live tools
- Birthdays
- Transcripts
- Reports
- Channel/role manager
- Polls
