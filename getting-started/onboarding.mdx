The first time you open a server in the dashboard, you'll be guided through initial setup.

During onboarding you will:

- **Select which features to enable** — choose the features you want active on your server.
- **Configure basic settings** — set your welcome channel, mod log channel, and other defaults.

<Note>
  You can change all of these settings later from the individual feature pages.
</Note>
