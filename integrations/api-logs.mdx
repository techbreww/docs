title: "API logs"
description: "View and debug API requests made by the dashboard."
icon: "list"
---

The API logs page shows a record of API requests made between the dashboard and Discord. Use it to debug issues or monitor bot activity.

Access the logs from **Integrations** > **API Logs** in the [dashboard](https://brewbot.techbrew.dev). Each log entry shows:

- Timestamp
- Endpoint
- HTTP method
- Status code
- Response time
